About Contenticious

Contenticious is a small and clean Mojolicious web app with additional tools to provide a smooth Markdown publishing workflow for you. It's developed as open source with one of the most free licenses (MIT License) to make it really easy for you to get your stuff done.

Contenticious lives on github. There you'll find

  • a wiki with useful things
    Feel free to contribute and help other users.
  • the latest Contenticious distribution as zip- or tarball
  • an issue tracker
    If you found something that might be a bug or you want to talk about a new feature you wish to see in the next release, this would be the best place to discuss it.
  • the source code
    Maybe you want to follow this repository to keep in touch if something changes.
